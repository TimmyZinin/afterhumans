# Послелюди / Afterhumans

> Narrative walker от первого лица. Episode 0 пилот будущего сериала.

## О проекте

**Послелюди** — серийный narrative-walker проект. Игрок идёт от первого лица через три локации: **Ботаника** (последний оазис живого), **Город** (стерильный мир Homo Deus по Harari), **Пустыня** (пески эстетики Dune). Диалоги в тоне Rick and Morty, TV-MA, русский язык. Прохождение Episode 0 — 10-15 минут.

## Технический стек

- **Engine:** Unity 6 LTS (6000.0.72f1)
- **Render pipeline:** URP
- **Target:** Standalone macOS (Apple Silicon)
- **Dialogue:** Ink (inkle) + ink-unity-integration
- **Assets:** Quixel Megascans, Poly Haven, Kenney, Quaternius, Mixamo
- **Distribution:** ad-hoc signed `.dmg` через лендинг на `timzinin.com/afterhumans/`

## Статус

Pre-production, Блок 1, Этап 1 (вселенная).
