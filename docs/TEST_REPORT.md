## Ограничения тестирования

**CU display unavailable**: Unity Metal renderer блокирует macOS compositor в windowed mode. Computer-use MCP не может делать скриншоты (screenshot → "CU display unavailable"). Workaround: CLI `screencapture -x -R`.

**Mouse look невозможен извне**: Cursor lock mode (`CursorLockMode.Locked`) не позволяет внешним инструментам (cliclick, osascript, Python Quartz CGEvents) управлять камерой. Unity читает raw mouse delta через `Input.GetAxisRaw("Mouse X/Y")`, что несовместимо с синтетическими событиями при заблокированном курсоре.

**Keyboard input нестабилен**: CU `hold_key` и osascript `key down/up` периодически не доходят до игры при потере фокуса окна.

   ```csharp
   if (Input.GetMouseButtonDown(0) && UnityEngine.Cursor.lockState != CursorLockMode.Locked)

       UnityEngine.Cursor.visible = false;
   }
   ```
