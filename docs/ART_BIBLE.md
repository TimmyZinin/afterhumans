## 1. Общая философия стиля

**Ключевая формула:** *stylized-realistic* — узнаваемо реальный мир, но намеренно упрощённый и усиленный. Не фотореализм (не потянет M1 8GB + URP). Не cartoon. Что-то между *Sable* и *Death Stranding*: реалистичная геометрия и материалы, но **гипер-выраженное освещение**, **упрощённые детали**, **сильная цветовая политика** по локациям.

**Главный инструмент впечатления — не количество полигонов, а освещение + палитра + пост-обработка.** На URP M1 8GB мы не сможем сделать сцену с миллионом объектов, но мы можем сделать сцену с 100 объектами, которая выглядит лучше сцены с 10 000 объектов плохо освещённых.

**Три принципа:**
1. **Каждая локация — одна сильная цветовая идея.** Если игрок может описать локацию одним цветом — мы победили.
2. **Свет важнее текстур.** Дешёвая модель в правильном свете красивее дорогой модели в плоском свете.
3. **Тишина важнее детализации.** Пустота работает, когда она намеренная. Не заполняй пустоты просто чтобы заполнить.

## 4. Освещение по локациям

### 4.1 Ботаника

**Directional Light:**
- Температура: 3200K (тёплый)
- Интенсивность: 1.2
- Угол: 25° над горизонтом (низкое закатное солнце через окна оранжереи)
- Тени: enabled, soft, shadow strength 0.6

**Дополнительные источники:**
- 2-3 Point Lights тёплого жёлтого (~2800K) — лампы на столах, над кофемашиной, в углу у Николая
- 1 Spot Light над серверной стойкой — холодный 5500K, маленький контраст с остальной тёплой сценой (намёк на то, что Ботаника — тоже немного data)
- Ambient: warm orange `#F5D8A3` с intensity 0.4

**Volumetric Fog (URP):**
- Включён, низкая плотность (density 0.015)
- Тёплый цвет: `#F5D8A3`
- Даёт ощущение пыльного воздуха в луче солнца через окно

**Key feel:** как будто ты пришёл в чьё-то очень тёплое лаборатория-квартиру в час до заката.

### 4.2 Город

**Directional Light:**
- Температура: 7500K (холодный дневной)
- Интенсивность: 0.8 (ниже чем в Ботанике — город более тусклый несмотря на свет)
- Угол: 45° (нейтральный, не закат)
- Тени: enabled, **очень чёткие, shadow strength 1.0** — придаёт стерильность

**Дополнительные источники:**
- Ambient light высокий: `#E0E8EF` с intensity 0.7 — заполняет всё плоским светом, убирает тёмные углы
- **Никаких тёплых источников** в городе. Принципиально.

**Volumetric Fog (URP):**
- Включён, плотность чуть выше чем в Ботанике (0.02)
- Холодный цвет: `#D8E0E8`
- Создаёт ощущение дистанции, пустоты, «воздух-вата»

**Key feel:** гипер-чистая больничная атмосфера, тишина, всё слишком ровно.

### 4.3 Пустыня

**Directional Light (солнце):**
- Температура: 2400K (очень тёплый, почти оранжево-красный)
- Интенсивность: **2.5** (HDR, закатное солнце яркое)
- Угол: 5° над горизонтом (почти касается горизонта)
- Тени: длинные, мягкие, shadow strength 0.9

**Sky / HDRI:**
- Poly Haven HDRI: `sunset_in_the_chalk_quarry_16k.hdr` или аналогичный закатный
- Skybox material: URP-совместимый HDRI
- Rotation подобрать так, чтобы солнце было в правильной стороне

**Ambient:**
- Тёплый охровый `#B87545` с intensity 0.5
- Fog тёплый, низкий: `#C97648` density 0.01

**Key feel:** безграничность, красота, закат который не двигается, тишина вдвоём с собакой.

## 5. URP Post-Processing Stack

Глобальный Volume + локальные per-scene Volumes. Один Global Volume Profile на сцену, одно Post Processing Layer на камере.

### Общие эффекты (все сцены)

| Effect | Value |
|---|---|
| **Bloom** | enabled. Intensity 0.5 (универсально). Threshold 1.0 |
| **Tonemapping** | ACES (cinematic look) |
| **Color Adjustments** | per-scene (см. палитры выше) |
| **Vignette** | слабое, intensity 0.2, roundness 0.8 |
| **Film Grain** | enabled, intensity 0.15, response 1.0 (лёгкая плёночная текстура для кинематографичности) |

### Ботаника (дополнительно)

| Effect | Value |
|---|---|
| **White Balance** | Temperature +15, Tint -5 (добавляем тепла) |
| **Color Adjustments** | Saturation +10, Contrast +5 |
| **Shadows, Midtones, Highlights** | Shadows cool (hex 6B7A85); Highlights warm (hex F5D8A3) |
| **Depth of Field** | focus 3m, aperture 5.6 (мягкий фон) |

### Город (дополнительно)

| Effect | Value |
|---|---|
| **White Balance** | Temperature -20, Tint 0 (прохладный) |
| **Color Adjustments** | Saturation **-30** (почти ч/б), Contrast +10 |
| **Chromatic Aberration** | 0.1 (намёк на glitch) |
| **Lens Distortion** | очень слабое, -0.05 |
| **Vignette** | intensity 0.3 (больше чем в Ботанике) |

### Пустыня (дополнительно)

| Effect | Value |
|---|---|
| **White Balance** | Temperature +30, Tint +10 (максимально тёплый) |
| **Color Adjustments** | Saturation +20, Contrast +20 |
| **Bloom** | Intensity **1.2** (солнце должно пылать) |
| **Depth of Field** | focus 15m, aperture 2.8 (длинные фокусы, эпичность) |
| **Motion Blur** | enabled, slight (для мягкой ходьбы по песку) |

## 6. Камера

### Общие параметры
- **FOV:** 65° (стандартный first-person, не слишком wide)
- **Height:** 1.65m (средний рост человека)
- **Near clip:** 0.1
- **Far clip:** 1000 (для пустыни нужны длинные дистанции)
- **Movement smoothing:** slight lag 0.1 (мягкость движения, не механическая)

### Head bob
- **Ботаника:** средний (естественно ходишь по знакомому месту)
- **Город:** минимальный (напряжение, аккуратный шаг)
- **Пустыня:** сильный (неровная поверхность песка) + случайные stagger-моменты

### Cinematic shots
Несколько заранее заданных камерных позиций, которые вызываются triggers:
- **Пробуждение в кресле**: камера в позиции сидящего игрока, медленно «просыпается» (fade from black + slight drift)
- **Первый выход из Ботаники**: в дверном проёме на 0.5 секунды фиксируется широкий shot коридора
- **Первый взгляд на Город**: перед выходом на улицу камера замедляется, показывая улицу в ширину
- **Первый взгляд на Пустыню**: при прохождении через ворота — slow pan справа налево по горизонту
- **Приближение к Курсору**: за 30м до курсора FOV медленно сужается с 65° до 50° (зум-эффект), создавая дистилляцию фокуса

## 7. Анимации

### FirstPerson controller
- Из **Unity Starter Assets FirstPerson**, встроенные idle/walk/run/jump-less
- Для нас: idle + walk + run, без jump (нам не нужен)

### NPC в Ботанике (Mixamo)
- **Саша**: `Sitting Laughing` loop + `Sitting Rubbing Arm` + `Sitting Thinking`. Рандомный выбор каждые 8-12 секунд.
- **Мила**: `Sitting at Desk` loop + `Thumbs Up` + `Reading` gesture. При разговоре — `Talking` gesture.
- **Кирилл**: `Standing` idle + `Cooking` (если найдётся) + `Look Over Shoulder`. Основная поза — руками у плиты.
- **Николай**: `Sitting Drinking` (если есть, иначе `Sitting Idle` + prop стакан в руке) + `Shaking Head` в момент определённых реплик.
- **Стас**: `Walking Backwards` + `Nervous Standing` + `Paranoid Looking Around` (или близкие). Постоянное движение — никогда не стоит на месте > 5 секунд.

### NPC в Городе (Mixamo)
- **Дмитрий**: `Walking` loop, бесконечное. Никаких gesture.
- **Анна**: `Sitting Idle` + при триггере Кафки → `Crying` или `Sad Idle` (для эмоционального момента).
- **Смотрительница**: `Standing Idle Arms Crossed`.
- **Ребёнок** (если есть): `Standing Idle` без движения.

### Кафка (главный вызов)
- Нужен квадруп-риг. Источники: Mixamo (нет квадрупов), Sketchfab CC0 (есть), Unity Asset Store (есть бесплатные dog models с анимациями).
- **Необходимые анимации:** `idle`, `sit`, `sit_down`, `stand_up`, `walk`, `run`, `bark`, `sniff`, `growl` (для сцены у сервера).
- **Optional:** `shake`, `scratch`, `lie_down`, `tail_wag`.
- **Приоритет при acquisition:** найти пак `Free Dog Animations` в Unity Asset Store или Sketchfab с полным набором.

## 8. Аудио

### 8.1 Общие принципы
- **Русский язык только** для диалогов (но диалоги — ТЕКСТ, не voice-acting, это экономит дни работы)
- **CC0 / freesound.org** для всех эффектов
- **Suno AI** (если Тим имеет доступ) для одной композиции в пустыне
- **NO voice acting** в Episode 0. Решение по финансам и времени.

### 8.2 Музыка

#### Ботаника
- **Жанр:** chill electronic ambient, lo-fi, тёплый
- **Референсы:** Nils Frahm, Emancipator, Bonobo (early), Tycho (spaces)
- **Темп:** 60-70 BPM
- **Источники:** Free Music Archive, Incompetech (Kevin MacLeod), ccMixter
- **Длина:** 4-6 минут loop
- **Ключ:** Am или Em (тёплые минорные)
- **Инструменты:** Rhodes-пиано, pad-synth, лёгкие пальчиковые хай-хэты

#### Город
- **Жанр:** ambient drone, почти тишина
- **Референсы:** Tim Hecker (минимальные моменты), Stars of the Lid, Eluvium
- **Темп:** без темпа, drone
- **Инструменты:** long sustained synth pad + occasional glitchy micro-sounds
- **Длина:** 4-5 минут loop
- **Ключ:** очень нейтральный, возможно просто sustained D

#### Пустыня
- **Жанр:** cinematic ambient с cellular/mid-east текстурами
- **Референсы:** Hans Zimmer *Dune (2021)* — «Ripples in the Sand», «Visions of Chani»; Jóhann Jóhannsson; Max Richter
- **Темп:** медленный, 50 BPM
- **Инструменты:** low brass-like drone + распевные женские голоса (без слов) + гулкий беззвучный барабан раз в 10-15 секунд
- **Длина:** 5-7 минут loop (игрок дольше идёт)
- **Ключ:** минор, возможно Dm с модальной окраской
- **Генерация:** Suno AI с промптом «cinematic desert ambient in style of Hans Zimmer Dune, slow, female wordless choir, low drone, 120 seconds»

#### Финальный экран (Credits)
- **Один долгий синтезаторный аккорд**, затухающий с титрами (~40 секунд)
- Простой sustain на том же Dm из пустыни

### 8.3 SFX

#### Ботаника
- Кофемашина (булькание, шипение)
- Тихий кашель Саши
- Шуршание страниц (Мила)
- Кипение турки (Кирилл)
- Звон стекла (Николай, стакан)
- Паяльник (Стас, редкие коротние звуки)

#### Город

- Тишина (важно: фоновый шум — это отсутствие шума)

## 9. UI и типографика

### Субтитры диалогов
- **Шрифт:** IBM Plex Sans (есть кириллица, читаемый, характерный). Backup: Inter или PT Sans.
- **Размер:** ~24pt в игре, динамически масштабируется под разрешение
- **Цвет:** `#FFFFFF` с 90% opacity
- **Background:** полупрозрачная тёмная плашка `#000000` 60% opacity
- **Положение:** нижняя четверть экрана, центр
- **Максимум строк:** 3
- **Speed:** typewriter effect ~30 chars/sec, можно пропустить нажатием E

### Выборы игрока
- **Шрифт:** IBM Plex Mono (моноширинный, подчёркивает «input в систему»)
- **Размер:** 20pt
- **Цвет:** `#FFFFFF` normal, `#F2C084` hover (тёплый — связь с Ботаникой)
- **Префикс:** `> ` перед каждой опцией (намёк на курсор в финале)
- **Положение:** под субтитрами

### Interaction prompt (нажать E)
- **Шрифт:** IBM Plex Mono
- **Текст:** `[E] говорить` / `[E] взять` / `[E] взаимодействовать`
- **Позиция:** возле объекта в мире (worldspace)
- **Цвет:** `#FFFFFF` с pulsing opacity 60%-100%

### Главное меню
- **Название игры:** IBM Plex Serif или Playfair Display (для заголовка, с характером)

## 10. Материалы и shaders

### Основной рендер
- **URP Lit Shader** для всех стандартных объектов (здания, пропсы, персонажи, Кафка)
- **URP Unlit Shader** для skybox, курсора, UI элементов в worldspace

### Shader Graph custom (если успеем)

1. **Cursor shader** — `> _` в воздухе
   - Unlit base + mask texture с ASCII
   - Slight glow через bloom
   - Mirror pulsing через time node
   - Опционально: subtle displacement noise для «парения»

2. **Glitch shader** (для редких моментов в Городе и у сервера в Пустыне)
   - Screen-space shader, activated per-frame через C# script
   - RGB split + vertical noise lines
   - Triggered by code, не always-on

3. **Sand shader** (для пустыни, если Megascans не даст достаточно красивое)
   - Base photoscanned texture + normal map
   - Height-based color gradient (top тёплый, tops оранжевее)

### Материалы Megascans
- Если используем Megascans — они приходят с готовыми PBR материалами (albedo, normal, roughness, metallic, AO)
- В URP они требуют адаптации: ручная конвертация материалов через `Edit > Rendering > Materials > Convert Selected Built-in Materials to URP`
- **Megascans Bridge для Unity** автоматизирует этот процесс при экспорте

## 11. VFX (Visual Effects)

### Ботаника
- **Пылинки в луче солнца** — URP Particle System, ~50 частиц, медленные, следуют свету через окно
- **Пар от кофемашины** — локальный particle emitter возле Кирилла (тихий, постоянный)
- **Мерцание серверной стойки** — 2-3 точечных света с легко анимированной intensity

### Город
- **Ничего лишнего.** Здесь визуальная чистота принцип. VFX = 0.

### Пустыня
- **Песок на ветру** — particle system, низкоплотный, по всей сцене, движение в направлении ветра
- **Eerie glitch particles у сервера** — редкие, только в радиусе 3м от сервера, маленькие RGB-искажения
- **Тень от курсора** — не VFX, а честная shadow projection (настроить курсор как shadow caster)
