# Action Plan — Ботаника → AAA

## Метод

Один цикл = код → build → screenshot → оценка Тима → fix → повтор.
Без спринтов. По одному изменению. Каждое проверяем.

| 1 | **Текстуры на пол/стены/мебель** | +3 | «Пол похож на пол? Стены на стены? Или всё ещё flat?» |

## Что делает Тим

После каждого `build → screenshot` я показываю результат. Тим отвечает одним из:

- **«норм, дальше»** → перехожу к следующему пункту
- **«хуже стало»** → откатываю, пробую иначе
- **«направление верное, но [конкретика]»** → фикшу по feedback
- **«открою Editor, покручу сам»** → жду скриншоты с пометками
